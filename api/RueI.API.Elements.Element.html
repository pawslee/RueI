Remarks

Elements are how you show text to a player using RueI. You can create an instance of a class that derives from Element, such as BasicElement or DynamicElement, and show the text to a player.

NoparseSettings

Remarks

This allows for custom behavior when parsing certain values in the text of the Element. This prevents players from bypassing noparse and breaking hints. The default value is ParsesNone. It is recommended to keep this value as the default.

Remarks

The GetParsedData(ReferenceHub) method is called when a Display is updated and RueI combines every Element that belongs to it. Implementations for this method need not call Parse(string, Element) every time this method is called; instead, the ParsedData can be saved and reused.
