Remarks

The BasicElement is a simple element with fixed text. To update the text of a BasicElement, you can send the player a new BasicElement with the same Tag.

Remarks

The GetParsedData(ReferenceHub) method is called when a Display is updated and RueI combines every Element that belongs to it. Implementations for this method need not call Parse(string, Element) every time this method is called; instead, the ParsedData can be saved and reused.
