Represents a unique identifier for an element within a RueDisplay. This class cannot be inherited.

Remarks

A RueDisplay can only have one element with a specific Tag at a time. If two Tags are created with the same string, they are considered equal. You can use the Tag() constructor to create a tag that will only be equal to itself.
